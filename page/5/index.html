让陌生人迅速相爱的36个聊天话题

@ · Dec 24, 2015 · 1 min read

第一组 第二组 第三组 转自：纽约时报 到了适婚年龄的我，为获得一份恋爱感到一份深深地苦恼。不算很笨的我但是，对于恋爱总是不得其法。我想会这样是因为多年在外求学和在家里的教育影响的。家里大人忙碌，十岁开始住校。在家为了不给大人们添忧，在外为了更好的保护自己，很多想法不愿与人分享，很多事情不愿去追究。慢慢地，忽略了人与人之间交流，忘了如何去与人沟通。独自走在人生的道路上。有幸看到了这三组聊天的话题，与那些与我有同样困难的伙伴分享，在打开你的话匣子的同时也开启你的恋爱旅程。 曼迪·莱恩·卡特隆(Mandy Len Catron)为“现代爱情”专栏写了一篇文章《如何快速与陌生人相爱》，她在文中提到心理学家阿瑟·亚伦(Arthur Aron)等人的研究成果：两个陌生人之间的亲密关系或许可以通过彼此询问一些特别的个人化问题而快速升温。这 36 个问题分为三组，一组比一组来得寻根究底。 这个理论的核心是，共同的脆弱能促进亲近感。这项研究的作者们称：“同伴之间发展亲密关系的关键模式在于持续、逐步升级、相互且个人化的袒露自我。”允许自己和另一个人共享脆弱可能非常困难，下面这个练习能迫使你做到这一点。 卡特隆和她的朋友在文中尝试的最后一件事是彼此对视四分钟，这个过程没有被详细记录（其实可以从两分钟到四分钟不等）。但是卡特隆毫不迟疑地推荐这件事。“两分钟的效果已经很惊人了，”她告诉我，“四分钟真的非常有效。” 第一组 如果可以在世界上所有人中任意选择，你想邀请谁共进晚餐？ 你想成名吗？想以什么方式成名？ 打电话之前你会先排练一下要说什么吗，为什么？ 对你来说，“完美”的一天是什么样的？ 你上次自己唱起歌来是在什么时候，给别人唱呢？ 如果你能活到 90 岁，同时可以一直保持 30 岁时的心智或身体，你会选择保持哪一种呢，心智还是身体？ 你是否曾经秘密地预感到自己会以怎样的方式死去？ 说出三件你和你的伴侣看上去相同的特征。 人生中的什么东西最令你感激？ 如果你能改变被抚养成人过程中的一件事，会是哪一件。 花四分钟时间，尽可能详细告诉伴侣你的人生经历。 如果你明天一觉醒来就能拥有某种才能或能力，你希望那会是什么能力呢？ 第二组 如果有一个水晶球可以告诉你关于自己、人生，未来乃至任何事情的真相，你会想知道吗？ 有没有什么事是你一直梦想去做而没有去做的，为什么没有做？ 你人生中最大的成就是什么？

@ · Dec 17, 2015 · 4 min read

HTTP 状态码的作用是：web 服务器用来告诉客户端，发生了什么事。 状态码位于 HTTP Response 的第一行中，会返回一个”三位数字的状态码“和一个“状态消息”。 ”三位数字的状态码“便于程序进行处理， “状态消息”更便于人理解。 状态码分类 HTTP 状态码被分为五大类， 目前我们使用的 HTTP 协议版本是 1.1， 支持以下的状态码。随着协议的发展，HTTP 规范中会定义更多的状态码。 小技巧： 假如你看到一个状态码 518， 你并不知道具体 518 是什么意思。 这时候你只要知道 518 是属于（5XX，服务器错误就可以了） 已定义范围 分类 状态码 范围 含义 1XX 100-101 信息提示 2XX 200-206 成功 3XX 300-305 重定向 4XX 400-415 客户端错误 5XX 500-505 服务器错误 常见的状态码 一般人只需要了解以下常见的状态码就够了， 如果你想了解更多， 请继续往下看。

延边——美味与情谊

@ · Oct 5, 2015 · 1 min read

延边，这个朝族与汉族混居的城市，早在初上大学的时候就出现在了我的版图中。同学的种种诉说中，吐露着它的美好。为了了却我一睹延吉的美，我改变了以前经常回家的习惯，拿出一部钱来到了这个中国东方小镇。 十月一日 这一天是假期的第一天，我晚上睡了大概三个小时左右的样子就拖起行李箱到北京站赶凌晨三点的火车。这不是我第一次凌晨三点起，但却是我第一次凌晨三点赶火车。坐上动车，我边开始了从北京到长春转车到 长春的行程。 火车一出河北，外面便开始下雨。车速很快，车窗上的雨滴向后滑去，并慢慢的向上升起。我在车里分不清雨下的大小，但是感觉到了温度在一点点的下降。抬头望向车里的提示牌，温度便已是个位数。但这并没有磨灭我对延边玩耍的火热的心情。 下午三点便已经到达延边州，雨还继续的下着，由风吹到身上的时候让我浑身打了个冷颤，口中哼起了的话的名歌——冷冷的冰雨在脸上胡乱的拍。原本孤单寂寞冷的心情，从我走出车站的那一刻起，便已经被站外的等待接我的同学感动到温暖如炉，已经感觉到眼中有一种东西在涌动。 晚上，同学请我吃了元奶奶紫菜包饭。店面不大，味道确实很好。用粟子叶一点点的在将肉包起来吃的味道很独特。对这种一边自己边吃，边包的过程体验的不亦乐乎。即使我的言语不多，在这种场合也不至于特别尴尬。 并不是所有的事情都是不尴尬的，晚上迎来了一件尴尬的事情，就是到处没有宾馆可以租住。面对这种情况，我同学开始自责起来。我并没有怪她，毕竟是自己策划的不周到，我也不是很责怪的我自己，这真的是我的第一个旅行，面对旅行需要学习的事情很多。 于是我在洗浴中心借宿了。 十月二日 雨不下了，延边依然大风。 我昨天在路途中有些劳累，今天便多睡了一些时候。和同学约好的八点见面也推迟了十点。 十点我来到了他们学校，参观了他们够长的校园。对于我来说，他们的学校很大，足球场很美，图书馆有够宽敞，校园背面还有还可以上山。这让我对在延大上学感到了舒适感，也深深的体会到了为啥有些人建议在选择大学的时候一定要去亲眼看一看，感受一下适合自己不？但是对于我们，没有机会进行很多的感受，只能走到那里，然后再进一步的走下去。 她也没有怪我，却是满脑子担心我没有机会到处走走。她这次的关心同样让我感受到了延边人民的极大热情。但她不知道，对于我来说，来到延吉每走一步对我来说都是一种旅行。有了她的陪伴，让这次旅行更加的意义非凡。 十月三日 受大雪风封山的影响，今天依然没有去长白山，但是去了品尝了泉州拌饭。泉州拌饭在当地十分有名，拌饭周围脆，将中间的饭拌到边上的一会也脆了。拌饭里加的菜也很新鲜。这种拌饭让我真心的喜欢。 十月四日 这一天我来到了长白山脚下，白雪盖山。山雪辉映，心中感慨祖国大好河山。而我在北京，心中仍为那一隅绞尽脑汁，为一人伤心难过。这都是对世界的辜负呀。 地下森林，无数的数目生长于地下，而在树间有一个直接迎向远方-希望。越远越开阔，开阔的那边是充满的美好。 晚上和武峰日的兄弟喝酒，喝的心情顺畅。
