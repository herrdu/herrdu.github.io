virtualenvwrapper的安装

@ · Oct 30, 2015 · 2 min read

安装 支持的 SHELL virtualenvwrapper 是在 Bourne Shell 兼容语法上定义的一个 Shell 函数的合集。在 OS X 和 Linux 系统上，可以用下面这些 Shell 进行自动化的测试： bash Ksh zsh 它也许可已运行在其他的 Shell 上，如果你发现了它能够运行除了以上列出来的其它 Shell 上，请告诉我。如果你在没有完全重写它而让他运行在其他的 Shell 上，你可以通过bitbucket project page发送一个 Pull 请求。如果你写了一个克隆的版本去适应不兼容的 Shell 的话，也请告诉我，我会在这个界面上加上一个链接。 Windows 命令提示符 David Marble 已经将 virtualenvwrapper 到能够在微软的 Windows 命令提示符上运行的 Windows 批处理脚本。这是也是一个独立的分布式的再实现。你可以通过 PyPI 下载 virtualenvwrapper-win MSYS 它可以使用 virtualenv 和本地的 Windows Python 安装文件封装MSYS。为了让它能够运行，你需要额外的定义一个环境变量，起名字为——MSYS_HOME,包括指向 MSYS 安装程序的根目录。 export WORKON_HOME=$HOME/.virtualenvs export MSYS_HOME=/c/msys/1.0 source /usr/local/bin/virtualenvwrapper.sh 或者 export WORKON_HOME=$HOME/.

延边——美味与情谊

@ · Oct 5, 2015 · 1 min read

延边，这个朝族与汉族混居的城市，早在初上大学的时候就出现在了我的版图中。同学的种种诉说中，吐露着它的美好。为了了却我一睹延吉的美，我改变了以前经常回家的习惯，拿出一部钱来到了这个中国东方小镇。 十月一日 这一天是假期的第一天，我晚上睡了大概三个小时左右的样子就拖起行李箱到北京站赶凌晨三点的火车。这不是我第一次凌晨三点起，但却是我第一次凌晨三点赶火车。坐上动车，我边开始了从北京到长春转车到 长春的行程。 火车一出河北，外面便开始下雨。车速很快，车窗上的雨滴向后滑去，并慢慢的向上升起。我在车里分不清雨下的大小，但是感觉到了温度在一点点的下降。抬头望向车里的提示牌，温度便已是个位数。但这并没有磨灭我对延边玩耍的火热的心情。 下午三点便已经到达延边州，雨还继续的下着，由风吹到身上的时候让我浑身打了个冷颤，口中哼起了的话的名歌——冷冷的冰雨在脸上胡乱的拍。原本孤单寂寞冷的心情，从我走出车站的那一刻起，便已经被站外的等待接我的同学感动到温暖如炉，已经感觉到眼中有一种东西在涌动。 晚上，同学请我吃了元奶奶紫菜包饭。店面不大，味道确实很好。用粟子叶一点点的在将肉包起来吃的味道很独特。对这种一边自己边吃，边包的过程体验的不亦乐乎。即使我的言语不多，在这种场合也不至于特别尴尬。 并不是所有的事情都是不尴尬的，晚上迎来了一件尴尬的事情，就是到处没有宾馆可以租住。面对这种情况，我同学开始自责起来。我并没有怪她，毕竟是自己策划的不周到，我也不是很责怪的我自己，这真的是我的第一个旅行，面对旅行需要学习的事情很多。 于是我在洗浴中心借宿了。 十月二日 雨不下了，延边依然大风。 我昨天在路途中有些劳累，今天便多睡了一些时候。和同学约好的八点见面也推迟了十点。 十点我来到了他们学校，参观了他们够长的校园。对于我来说，他们的学校很大，足球场很美，图书馆有够宽敞，校园背面还有还可以上山。这让我对在延大上学感到了舒适感，也深深的体会到了为啥有些人建议在选择大学的时候一定要去亲眼看一看，感受一下适合自己不？但是对于我们，没有机会进行很多的感受，只能走到那里，然后再进一步的走下去。 她也没有怪我，却是满脑子担心我没有机会到处走走。她这次的关心同样让我感受到了延边人民的极大热情。但她不知道，对于我来说，来到延吉每走一步对我来说都是一种旅行。有了她的陪伴，让这次旅行更加的意义非凡。 十月三日 受大雪风封山的影响，今天依然没有去长白山，但是去了品尝了泉州拌饭。泉州拌饭在当地十分有名，拌饭周围脆，将中间的饭拌到边上的一会也脆了。拌饭里加的菜也很新鲜。这种拌饭让我真心的喜欢。 十月四日 这一天我来到了长白山脚下，白雪盖山。山雪辉映，心中感慨祖国大好河山。而我在北京，心中仍为那一隅绞尽脑汁，为一人伤心难过。这都是对世界的辜负呀。 地下森林，无数的数目生长于地下，而在树间有一个直接迎向远方-希望。越远越开阔，开阔的那边是充满的美好。 晚上和武峰日的兄弟喝酒，喝的心情顺畅。

《岛上书店》读后感

@ · Aug 16, 2015 · 1 min read

偶尔在三联书店上遇见了《岛上书店》这本书。一见到它那古红色的店门和充满故事的书名吸引，于是默默的买下来此书。 此书讲的是岛上一个书店里发生的故事，是一个先悲后喜，然后归于平淡的故事。对于习惯看美国大片的我们来说，这本书起初读起来有点索然无味。 开篇新来的出版商推销员遇到了丧妻的性格诡异的书店老板，这本来没有什么值得惊喜的故事，遇到一点点挫折是在正常不过的了。而老板呢？好象是祸不单行，不仅丧妻还丢失了可以让自己退休的珍藏的书籍和门口多出来的一个弃婴。 慢慢都下来，确实觉得虽然平淡，但处处透漏着生活中点点滴滴发生的故事。故事的发展自然，逼近现实；然而每一步进展都埋伏着转折，透漏着人生的神奇之处——人生处处有惊喜，永远不知道下一步如何发展。 老板书店丧妻痛苦中与推销员相恋，获得了新的幸福生活；又拾得弃婴得到一个聪明富有天分的女儿，能够重新收获幸福的生活。即使后来身患癌症，珍藏书籍的失而复得恰好弥补了医疗费用。书店老板的一生幸福悲伤的交替出现，平淡的描写手法，让我们感受到了一种坚强，一种因果循环的结果。给予在平凡中的我们对生活中的启示与指引。 此外本书被评为现象级书籍，让我很是疑惑。这类评价之前从未接触过，但是查阅之后发现被评为现象级条件有三： 要有过程 社会公认 产生影响 我想人们给予此书这样大的评价，也是对作品的充分肯定吧。而这部作品也让我见识到了一种新的写作风格。
