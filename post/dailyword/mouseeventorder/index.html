每日一词：MouseEvent Order

@Herr Du · Jul 25, 2019 · 1 min read

上篇文章，我们关注了 MouseEvent 这个接口，但是它还有一个有趣的定义我们没有关注到，就是 MouseEvent 产生的 Order。 我们关注 MouseEvent 产生顺序的主要一下四种场景：

同一个元素上事件触发的顺序
Event Type Element Notes
mouseMove
mouseOver A Pointing device is moved into element A…
mouseenter A
mousemove A Multiple mousemove events
mouseout A Pointing device is moved out of element A…
mouseleave A
B 是 A 与子元素的情况
Event Type Element Notes
mousemove
mouseover A Pointing device is moved into element A…
mouseenter A
mousemove A Multiple mousemove events
mouseout A Pointing device is moved into nested element B…
mouseover B
mouseenter B
mousemove B Multiple mousemove events
mouseout B Pointing device is moved from element B into A…
mouseleave B
mouseover A
mousemove A Multiple mousemove events
mouseout A Pointing device is moved out of element A…
mouseleave A
元素重叠的状况

有的时候，会因为 Css 的操作造成元素的重叠如下：

C 是 B 的子元素，B 是 A 的子元素，A、B、C 在同一个位置重叠在一起。

Event Type Element Notes
mouseMove
mouseOver C Pointing device is moved into element C, the topmost element in the stack
mouseEnter A
mouseEnter B
mouseEnter C
mouseMove C Multiple mousemove events
mouseout C Pointing device is moved out of element C…
mouseleave C
mouseleave B
mouseleave A
