Play环境在mac上的搭建

@ · Sep 14, 2015 · 1 min read

  1. 安装 java jdk

Mac 上本来内置 java jdk ，但是是 1.6 版本的。这里我们安装 java1.7，下面是下载地址:java1.7

  1. 安装 activator

与 java1.7 想兼容的 play Framework 版本是 2.3.x 的，因此我们下载 2.3.x 的，下载地址：https://www.playframework.com/documentation/2.3.x/Home

安装的步骤参考：官网上提供的“Installing Play”章节，下载 activator 之后，添加导环境变量即可。

添加方法：在 .bash_profile 中添加 export PATH=\$PATH:/relativePath/to/activator 类似语句即可。

进入到项目路径，输入 activator，程序自动更新 catch 包，更新完毕之后，输入 run 即可运行项目。

  1. Mongo 数据库

找到适合 Mac 版的 mongo 数据库，安装上即可，配置好环境变量，然后运行 mongod 启动数据库

Mongo 数据库图形管理工具：Mongohub

下载地址：https://github.com/jeromelebel/MongoHub-Mac

4.安装 Tomcat

  • 版本：tomcat-8.0
  • 安装方法 + 下载地址 http://tomcat.apache.org/download-80.cgi + 解压重命名为 tomcat，然后将文件夹复制到/Library 下面，路径为/Library/tomcat + 为 tomcat 添加环境变量 到.bash_profile,语句为 export PATH=\$PATH:/Library/tomcat/bin + 添加 tomcat 的快速启动 ，建立 tomcat 文件 输入下面的语句,然后放到/usr/bin 目录中



   #!/bin/bash
  case $1 in
  start)
  sh /Library/tomcat/bin/startup.sh
  ;;
  stop)
  sh /Library/tomcat/bin/shutdown.sh
  ;;
  restart)
  sh /Library/tomcat/bin/shutdown.sh
  sh /Library/tomcat/bin/startup.sh
  ;;
  *)
  echo “Usage:start|stop|restart”
  ;;
  esac
  exit 0

  • tomcat 文件配置 + 在/Library/tomcat/conf/Catalina/localhost 目录中添加 misc-admin.xml 文件，然后输入下面语句： 
    <Context path=“/misc-admin” docBase=“/Volumes/duyutao/work/trunk/pinclub/misc-admin” reloadable=“true”/>
