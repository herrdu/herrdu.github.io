virtualenvwrapper 的常规应用

@ · Nov 27, 2015 · 1 min read

  1. #### virtualenvwrapper 简介 virtualenvwrapper 是 python 的一个虚拟环境管理工具。它是对 virtualenv 工具的一个扩展。
  2. #### 功能
    • 在一个地方管理所有的虚拟环境
    • Wrappers 帮助你管理虚拟环境（创建，删除，复制）.
    • 一个简单的命令，在虚拟环境之中随意切换.
    • 命令可以由 Tab 补齐，虚拟环境是命令的参数
    • 所有的操作者都可以有自己的用户配置文件
    • 有更多的创建共享扩展的系统插件
  3. #### 安装与创建一个项目


   $ pip install virtualenvwrapper
   …
   $ export WORKON_HOME=~/Envs
   $ mkdir -p $WORKON_HOME
   $ source /usr/local/bin/virtualenvwrapper.sh
   $ mkvirtualenv env1

  1. #### 常用命令
    • ###### lssitepackages 查看虚拟环境中安装的 package
    • ###### ls \$WORKON_HOME 查看已经存在的虚拟环境和其他的配置
    • ###### workon 显示后者切换虚拟工作环境 例子：workon env1
    • ###### echo \$VIRTUAL_ENV 显示当前虚拟环境的目录
    • ###### deactivate 退出虚拟环境
    • ###### rmvirtualenv 删除虚拟环境
