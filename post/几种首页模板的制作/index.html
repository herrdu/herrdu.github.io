1. Play 和 scala 形成的模板

第一、制作 Header 和 footer 界面的代码片段，以方便填充到代码 main 文件里面


Header代码：
@(admin: AdminUser)
@import java.text.SimpleDateFormat;
<div id="navbar" class="navbar navbar-default navbar-fixed-top">
    //省略的代码
</div>



Footer代码：
@(source:String)
<div id="footer" class="@source">
    //省略的底部代码
</div>

第二、制作 main.scala 页面，这是一个有 Html 完整结构的文档，包括：Doctype 声明、header、body。头部使用 scala 语法传入用到的变量和需要插入的代码片段


@(title: String = "后台管理系统")(keywords: String = "后台管理系统")(description: String = "后台管理系统")(style: Html)(content: Html)(javascript: Html)

<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml" lang="zh-cn" xml:lang="zh-cn">
<head>
    <meta charset="UTF-8">
    <title>@title</title>
    <meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1">
    <meta name="keywords" content="@keywords">
    <meta name="description" content="@description">
    <script type="text/javascript" src="@assets.AdminCdnAssets.url("ued.import.js")" charset="utf-8"></script>
    @style
    <!--[if lte IE 9]>
        <link rel="stylesheet" href="@assets.AdminCdnAssets.root("skin/ace-part2.css")" class="ace-main-stylesheet" />
        <![endif]-->
    <!--[if lte IE 9]>
         <link rel="stylesheet" href="@assets.AdminCdnAssets.root("skin/ace-ie.css")" />
    <![endif]-->
    <!--[if lte IE 8]>
        <script src="@assets.AdminCdnAssets.root("js/widget/html5shiv.js")">

        <script src="@assets.AdminCdnAssets.root("js/widget/respond.js")">
    <![endif]-->

</head>
<body class="skin-1">
@content
@javascript
<div id="confirmDialog" class="hide">
</body>
</html>
